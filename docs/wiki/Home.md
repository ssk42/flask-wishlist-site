# Family Wishlist Wiki

Welcome to the Family Wishlist project wiki! This is a central hub for documentation regarding the application's architecture, development, features, and deployment.

## Project Purpose
To provide a beautiful, modern web application for families to share wishlists and coordinate gift-giving while keeping surprises intact.

- [**Development**](Development.md): workflows, testing, and coding standards.
- [**Features**](Features.md): Detailed explanation of application features.
- [**Deployment**](Deployment.md): Instructions for deploying the application to Heroku.
